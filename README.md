## 📦 주요 기능
**1️⃣ 소셜 로그인 기능**
- 카카오톡, 구글, X 로그인을 지원

**2️⃣ 디톡스 시간 조회**
- 홈 화면에서 조회
- 총 누적 시간에 따라 커지는 행성을 통해 직관적으로 구분 가능
- 오늘 누적 시간을 별도로 표시

**3️⃣ 디톡스 시간을 이용한 경쟁**
- 레이스 화면에서 모든 유저의 누적 시간을 1위부터 100위까지 표시
- 자신의 순위를 확인 가능

**4️⃣ 앱 사용량 조회 및 허용 어플 관리**
- 마이페이지에서 앱 사용량과 트로피 정보를 확인 가능
- 디톡스 중에 사용할 허용 앱을 설정 가능.
- 허용 앱의 이용 시간을 설정 가능(기본 30분).

**5️⃣ 디톡스 시간 측정**
- 타이머를 시작해 디톡스 시간을 측정.
- 디톡스가 진행 중일 때는 다른 화면으로 이동할 수 없음.
- 디톡스 진행 중에 마이페이지에서 설정한 허용 앱 사용 가능. 
이용 시간을 모두 소모한 경우에는 앱 사용 불가.
